По активному профилю.

Топ видео — YouTube
Откуда возьмутся реальные цифры?

Сейчас данные пустые — это правда. Чтобы появились настоящие метрики, нужно:

  • YouTube — добавить scope youtube.readonly и дёрнуть /youtube/v3/channels?part=statistics&mine=true + /youtube/v3/videos для топа.
  • TikTok — добавить scope video.list и research.adlib.basic; данные через /v2/research/video/query/. Для просмотров своего аккаунта — /v2/video/list/.
  • Instagram / Facebook — есть в текущих скоупах: /{ig-user-id}/insights и /{page-id}/insights.

Если хочешь — скажи «подтяни статистику», и я добавлю scopes + endpoints.
